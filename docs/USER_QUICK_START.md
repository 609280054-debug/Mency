# 用户快速上手

完整说明请看：

- [中文安装与使用说明](INSTALL_USAGE_CN.md)
- [English installation and usage guide](INSTALL_USAGE_EN.md)

最短流程：

1. 下载 `AI-Tuning-Rack-win-x64.zip`
2. 解压 ZIP
3. 双击 `AI Tuning Rack.exe`
4. 在右侧模型设置里填写自己的 API Key
5. 上传干声 WAV 和伴奏 WAV，或者开启实时监听
6. 写一句声音目标
7. 点击生成完整调试方案

新手建议优先使用“上传干声 + 上传伴奏”的方式，比实时监听更稳定。
