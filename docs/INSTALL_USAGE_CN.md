# Mency AI Tuning Rack 中文安装与使用说明

这份说明写给第一次使用这类工具的朋友。你不需要会编程，只要会下载、解压和填写 API Key 就可以开始用。

## 1. 这个软件是做什么的

Mency AI Tuning Rack 是一个 Windows 桌面调音辅助工具。

你可以把自己的干声和伴奏交给它分析，也可以让它实时监听你在宿主或机架里播放的声音。它会根据声音状态和你的目标风格，生成一套接近专业调音师流程的建议，包括：

- 应该先做哪些处理
- 推荐使用什么类型的插件
- 每个插件的参数从哪里开始调
- 试唱时要重点听什么
- 如果太闷、太刺、太空、太压，下一步怎么改

它不会自动替你控制 Studio One、Cubase、FL Studio 或 Ableton。它给的是可以照着做的调音方案。

## 4. 准备 API Key

这个软件需要连接大模型才能生成完整调音方案。公开版不包含我的付费 Key，也不会消耗我的额度。

你需要准备自己的 API Key。默认推荐 DeepSeek，也支持其他 OpenAI-compatible 接口。

### DeepSeek 怎么填

如果你用 DeepSeek，一般这样填：

```text
Base URL: https://api.deepseek.com
Model: deepseek-v4-pro
API Key: 粘贴你自己的 DeepSeek API Key
```

如果模型名以后有变化，以 DeepSeek 控制台提供的模型名为准。

### 其他模型能不能用

可以，只要它兼容 OpenAI API 格式，通常就能填。

常见需要填写三项：

```text
Base URL: 模型服务商给你的接口地址
Model: 模型名称
API Key: 你的 API Key
```

例如 OpenAI、Qwen、Kimi、OpenRouter、硅基流动等，思路都是一样的。

## 5. 第一次配置软件

打开软件后，按这个顺序做：

1. 在右侧找到模型设置区域
2. 如果只用 DeepSeek，直接粘贴 API Key
3. 如果用其他平台，展开高级模型设置
4. 填写 Base URL、Model、API Key
5. 点击保存

保存后，软件会把配置写入你电脑本地。这个配置不会上传到 GitHub。

## 6. 选择宿主或使用场景

左侧或中间区域可以选择目标宿主。

如果你自己用 Studio One 8，可以选 Studio One 8 或 Studio Pro 8 对应项。

如果别人用其他软件，可以选择：

- Studio One 6
- Studio One 7
- Cubase / Nuendo
- FL Studio
- Ableton Live
- Generic VST Chain

不确定选哪个时，就选 `Generic VST Chain`。它会给出更通用的插件链建议。

## 7. 方法一：上传干声和伴奏

这是最简单、最稳的方式。

你需要准备两个 WAV 文件：

- 干声：只包含人声，不要带混响、压缩、伴奏
- 伴奏：你的歌曲伴奏或 beat

操作步骤：

1. 点击干声上传区域，选择干声 WAV
2. 点击伴奏上传区域，选择伴奏 WAV
3. 点击分析录音
4. 等待分析结果出现
5. 填写你的声音目标
6. 点击生成完整调试方案

声音目标可以写得很口语化，例如：

```text
我想要温柔贴耳一点，不要太刺，适合流行男声。
```

或者：

```text
想要更亮、更靠前，但是不要齿音太重。
```

## 8. 方法二：实时监听

实时监听适合你不想导出音频，想直接在宿主、机架或 DAW 里播放给软件听。

大致流程：

1. 在软件里选择一个输入设备
2. 点击开始监听
3. 在 Studio One、机架或 DAW 里播放你的录音
4. 让软件看到实时电平和分析数据
5. 再点击生成调试方案

如果软件听不到 DAW 的声音，通常不是软件坏了，而是声音没有路由到输入设备。

你可能需要：

- 使用虚拟声卡
- 使用声卡的 Loopback 功能
- 把 DAW 输出路由到一个可被监听的输入
- 或者直接改用上传 WAV 的方式

## 9. 生成方案后怎么用

软件给出的方案通常会按专业流程排列，例如：

1. 清理噪声和低频
2. 修音或音高校正
3. EQ 调整
4. 压缩控制动态
5. 齿音处理
6. 饱和或染色
7. 混响和延迟
8. 与伴奏融合
9. 最终检查

你不需要一次把参数调到完美。建议先按方案给的参数作为起点，然后边听边微调。

## 10. 常见反馈怎么写

如果第一版方案不满意，可以把听感直接告诉软件。越具体越好。

可以这样写：

```text
人声太闷了，想再亮一点。
```

```text
齿音有点刺，s 和 ch 太明显。
```

```text
压缩太重，听起来不自然。
```

```text
混响太多，人声离我太远。
```

```text
人声和伴奏贴不住，像浮在外面。
```

软件会根据你的反馈给下一轮参数调整建议。

## 11. 常见问题

### 打开是黑屏怎么办

请确认你下载的是最新 Release。当前版本已经修复过打包后黑屏问题。

如果还是黑屏，可以重新解压一次，不要直接在 ZIP 压缩包里面运行 exe。

### 软件提示模型不可用怎么办

优先检查三件事：

1. API Key 有没有填错
2. Base URL 有没有填错
3. Model 名称是不是服务商支持的模型

### 生成方案很慢怎么办

这通常和模型服务商速度、网络、音频长度有关。可以先用较短的干声片段测试，例如 20 到 40 秒。

### 我的 Key 会不会被别人用

不会。公开安装包里没有你的 Key。你在自己电脑里填写的 Key 只保存在本地。

### 我不会路由实时监听怎么办

先用“上传干声和伴奏”的方式。这个方式最适合新手，也最稳定。

## 12. 推荐的新手流程

第一次使用时，建议这样做：

1. 下载安装器
2. 安装软件
3. 从桌面或开始菜单启动
4. 填 DeepSeek API Key
5. 上传一段 20 到 40 秒的干声 WAV
6. 上传对应伴奏 WAV
7. 写一句声音目标
8. 生成方案
9. 回到 DAW 里照着插件链一步步调
10. 把听感问题再反馈给软件继续微调
