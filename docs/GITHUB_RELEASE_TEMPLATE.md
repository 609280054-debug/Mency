# Mency AI Tuning Rack v0.1.0 Preview

First public Windows preview of Mency AI Tuning Rack.

## Download

Download `AI-Tuning-Rack-win-x64.zip`, extract it, then run:

```text
AI Tuning Rack.exe
```

## Included

- Windows desktop app
- Built-in packaged Python backend
- Real-time audio input monitoring
- Dry vocal WAV + backing WAV analysis
- AI tuning plan generation
- OpenAI-compatible model API settings
- DeepSeek default configuration
- Studio One 6/7/8 and generic VST workflow targets
- New Mency high-tech app icon
- Chinese and English installation guides

## Notes

- This preview build is unsigned, so Windows may show a warning on first launch.
- The public package does not include any paid API key.
- Each user needs to enter their own model API key inside the app.
- The app gives tuning suggestions; it does not automatically control your DAW or plugins.
