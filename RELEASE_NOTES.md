# Release Notes

## v0.1.0 Preview

First Windows portable preview build.

### Included

- Windows desktop app UI
- Packaged Python backend executable
- Real-time input device selection
- Real-time Peak, RMS, mud, sibilance, clarity, and transient-risk analysis
- Dry vocal WAV + backing WAV file analysis
- Target workflow choices for Studio One 6/7/8, Cubase / Nuendo, FL Studio, Ableton Live, and generic VST chains
- OpenAI-compatible model API configuration
- DeepSeek default model configuration
- Full AI-generated tuning-plan workflow
- Mency high-tech app icon
- Chinese and English installation and usage guides

### Known Limits

- The app is unsigned, so Windows may show an unknown-publisher warning.
- WAV analysis is the safest file-based workflow for this preview.
- The app outputs tuning suggestions only; it does not automatically control DAWs, racks, or plugins.
- Real-time monitoring depends on the user's Windows audio routing.
- API keys are stored locally and are not included in the public repository or release package.
